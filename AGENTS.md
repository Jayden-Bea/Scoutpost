# AGENTS.md

## Scope
These instructions apply to the entire repository.

## Project conventions
- Use Python 3.11+.
- Manage dependencies with `uv`.
- Keep code in a `src/` layout.
- Prefer small, well-tested modules.

## Validation commands
- `uv sync`
- `uv run pytest`
- `uv run streamlit run app.py`
